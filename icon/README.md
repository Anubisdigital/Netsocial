# Moles World App Icons

For the PWA to work correctly, you need the following icon sizes:

## Required Icon Sizes:
- icon-72x72.png (72x72 pixels)
- icon-96x96.png (96x96 pixels)
- icon-128x128.png (128x128 pixels)
- icon-144x144.png (144x144 pixels)
- icon-152x152.png (152x152 pixels)
- icon-192x192.png (192x192 pixels) - **Most important for Android**
- icon-256x256.png (256x256 pixels)
- icon-384x384.png (384x384 pixels)
- icon-512x512.png (512x512 pixels) - **Most important for splash screen**

## Icon Design Tips:
1. Use a clear image of a mole
2. Simple background (preferably solid color)
3. Make sure the mole is centered
4. Use high contrast for visibility
5. For maskable icons (Android adaptive icons), leave safe zone (â…™ of icon size from edges)

## Quick Generation:
You can use these tools to generate all icon sizes from one 512x512 image:
1. https://www.pwabuilder.com/imageGenerator
2. https://realfavicongenerator.net/
3. https://app-manifest.firebaseapp.com/

## App Icon Name:
Save your main app icon as: `mole-app-icon.png`

## Splash Screen Image:
Save your splash screen as: `mole-splash-image.png`
- Recommended size: 1920x1080 pixels
- Landscape orientation
- Feature a mole in its natural habitat
